# Hangman Game (Python)

A simple command-line Hangman game built using Python. The player guesses letters to uncover a hidden word before running out of lives.

---

## 📌 Features
- Random word selection
- Letter-by-letter guessing
- Tracks remaining lives
- Displays progress with underscores (_ _ _)
- Win/Loss conditions
